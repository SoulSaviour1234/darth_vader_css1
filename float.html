Float and Clear Property

The float property is used for positioning and formatting content, for example, letting text wrap around images. The clear property is used to control the behavior of floating elements.

  • float: left; - Floats element to the left.
  • float: right; - Floats element to the right.
  • clear: left; - Prevents left floats.
  • clear: right; - Prevents right floats.
  • clear: both; - Prevents both sides from floating.
